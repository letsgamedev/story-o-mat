# story-o-mat
Analyse texts and make predictions in the editor.

## Setup

### System

System requirements:

```
# Linux
apt install python3-tk
```

### Venv

Create and activate:

```
python3 -m venv venv
source venv/bin/activate
# Leave venv with 'deactivate'
```

Install project requirements:

```
pip install -r requirements.txt
```
